# J-FlashCards

JLPT N5~N1 단어/문법 플래시카드 iPhone 앱 (SwiftUI + SwiftData).

## 목표(MVP)
- 로컬(온디바이스) 데이터 저장
- 오늘의 단어
- 간단 퀴즈(뜻 맞추기)
- 매일 로컬 알림

## 개발 환경
이 레포는 **Xcode(맥)** 에서 빌드/실행하는 iOS 앱이야.

### 추천 워크플로우
- (Mac) Tuist로 Xcode 프로젝트 생성 → 빌드

## 데이터 소스
- 후보: https://www.jlptcode.com/
- **주의:** 데이터 크롤링/재배포는 사이트 이용약관/라이선스에 따라 제한될 수 있어.
  - 우선은 `Data/` 아래에 **수동으로** JSON/CSV를 넣고 앱에서 임포트하는 방식으로 시작하자.
